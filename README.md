## Run Locally

Clone the project

```bash
  https://github.com/ardwiinoo/dcd-bdpw-fe-calculator.git
```

Go to the project directory

```bash
  cd dcd-bdpw-fe-calculator
```

Run project
```bash
  localhost/dcd-bdpw-fe-calculator/index.html
```
